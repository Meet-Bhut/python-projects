# 🏦 Banking Management System

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Status](https://img.shields.io/badge/Status-Completed-success)
![Storage](https://img.shields.io/badge/Storage-JSON-orange)
![Platform](https://img.shields.io/badge/Platform-Console-lightgrey)

A **console-based Banking Management System** developed using **Python**. This project simulates the core functionalities of a real banking system, including customer account management, secure authentication, money transactions, transaction history, and an administrator panel.

The project follows **modular programming** principles and stores all account information permanently using **JSON**, ensuring that data persists even after the application is closed.
