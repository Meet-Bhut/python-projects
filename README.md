# ⭐ Featured Project

## 11. Banking Management System

The most advanced project in this repository.

A complete console-based banking system built using Python with modular programming and persistent JSON storage.

### Customer Features

- Create Account
- Secure Login
- Deposit & Withdraw
- Money Transfer
- Transaction History
- Change PIN
- Delete Account

### Admin Features

- Admin Login
- View All Accounts
- Search Accounts
- Unlock Accounts
- Delete Accounts
- Bank Statistics

### Security

- 4-digit PIN Authentication
- Account Locking after 3 failed attempts
- Unique Account & Transaction IDs
- Persistent JSON Data Storage
