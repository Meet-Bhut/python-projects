# Password Manager

A Python-based Password Manager that allows users to securely store and manage website passwords using file handling.

## Features

* Add Password
* Show Passwords
* Search Password
* Edit Password
* Delete Password
* File-based storage
* Menu-driven interface
* Input validation

* Functions
